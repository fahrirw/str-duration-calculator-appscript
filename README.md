# STR Duration Calculator (Google Apps Script)

## Description
A dynamic web-based tool to calculate STR duration based on cluster and input date using Google Apps Script and Google Sheets.

## Preview
![images](output.png)

## Features
- Dynamic calculation based on selected cluster
- Date-based logic for retrieving closest data
- Automatic STR date generation
- Form validation and error handling
- Interactive UI with SweetAlert

## Tools & Technologies
- Google Apps Script
- HTML, CSS, JavaScript
- Google Sheets

## Use Case
Used internally to simulate and calculate STR duration based on historical data stored in spreadsheets.

## Impact
- Automates manual STR calculation process
- Improves accuracy in date-based calculations
- Reduces time needed for data lookup
